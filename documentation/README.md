# IDPA – Webshop mit REST-API

Dieses Repository enthält die Projektarbeit im Rahmen der **Individuellen Praktischen Arbeit (IDPA)** an der Kantonsschule Hottingen.

## Projektübersicht

Im Rahmen der IDPA wird ein **webbasierter Webshop mit REST-API** entwickelt.  
Der Webshop bildet typische E-Commerce-Funktionalitäten ab (Produktkatalog, Warenkorb, Bestellungen) und kommuniziert über eine klar strukturierte REST-Schnittstelle zwischen Frontend und Backend.

## Projektziele

- Entwicklung eines funktionsfähigen Webshops
- Umsetzung einer REST-API nach gängigen Standards
- Klare Trennung von Frontend, Backend und Datenhaltung
- Persistente Speicherung der Daten
- Benutzerfreundliche Oberfläche
- Nachvollziehbare technische Dokumentation

## Projektorganisation

Das Projekt wird nach der Projektmanagement-Methode **IPERKA** durchgeführt.  
Planung, Rollenverteilung (PSP) sowie eine grobe Zeitplanung (Gantt-Diagramm) werden im Verlauf des Projekts erarbeitet und dokumentiert.
